## 项目简介

FRMT Worlds（Faramita Worlds）是一款本地化 AI 跑团平台，聚焦三件事：

1. 管世界：世界书、角色、章节、交互规则统一管理。
2. 跑剧情：AI 生成结构化叙事并驱动剧情推进。
3. 保连续：多层记忆、状态追踪、存档恢复保证长期体验稳定。

它不是单纯聊天窗口，而是一套可长期维护的叙事工作台。

## 核心特性

1. Agent Workflow 叙事链路：从记忆组装到后置审查，全流程可追踪。
2. 运行时世界书：卡片支持激活规则、注入位置、预算与时态效果。
3. 结构化叙事输出：环境、对话、交互判定、世界更新、建议动作。
4. 世界状态安全机制：字段级 delta、冲突检测、高风险延迟提交。
5. 文本导入向导：txt/md 解析、批量抽取、自动修复、去重与建议。
6. 本地多存档：按世界维度保存、加载、删除与自动存档。

## 应用场景

1. 单人文字跑团。
2. 世界观与角色设定创作。
3. 剧情企划打样与分支演练。
4. AI 叙事引擎研发与迭代验证。

## 技术栈

1. Electron
2. Vue 3 + TypeScript
3. Vite / electron-vite
4. Pinia
5. Dexie（IndexedDB）

## 快速开始

### 环境要求

1. Node.js 18+
2. npm 9+

### 安装依赖

```bash
npm install
```

### 本地开发

```bash
npm run dev
```

### 构建

```bash
npm run build
```

### Windows 打包

```bash
npm run dist:win
```

## 项目结构

```text
src/
	main/        # Electron 主进程
	preload/     # 预加载与 IPC 桥接
	renderer/    # Vue 应用与业务逻辑
	shared/      # 前后进程共享类型
resources/     # 演示资源与图标
saves/         # 本地存档样例
```
