# 活跃信息增强、投骰动效优化与编辑器功能补全方案

本方案将深度优化活跃信息展示、完善人物卡编辑字段，并调整投骰区的交互动效。

## 1. 活跃信息与人物卡详情 (ActiveCodex.vue)
*   **点击交互**：点击人物卡可展开显示详细属性（姓名、种族、年龄、性别、职业、等级及六维属性）。
*   **秘密词条过滤**：在活跃信息中自动过滤 `race` 和 `tags` 中带有 `(?)` 前缀的词条（如 `(?)至高神性` 将不显示），但在编辑器中仍保持可见。
*   **布局修正**：重新调整版面样式，解决右侧区域超出可见范围的问题，确保纵向排布整齐且不溢出。

## 2. 投骰区动效优化 (CodexZone.vue)
*   **动效时长**：将投骰区的进入（展开）和退出（收起）动效时间统一调整为 **2秒**，使其过渡更加自然平滑。

## 3. 世界卡编辑器功能补全 (CardEditor.vue & Interface.ts)
*   **背景与标签编辑**：在人物卡编辑区增加 `background` (背景) 和 `tags` (标签) 的编辑功能。
*   **多词条支持**：将 `background` 统一为数组存储，支持在编辑界面动态增加、删除多个背景词条。
*   **数据模型更新**：更新 [Interface.ts](file:///d:/Education/Project/Faramita%20Worlds/src/shared/Interface.ts) 中的 `CharacterCard` 定义，支持数组形式的背景。

## 4. 关键代码变更点
*   **`ActiveCodex.vue`**: 实现 `selectedCharId` 状态切换，并添加词条过滤函数。
*   **`CodexZone.vue`**: 修改 CSS `transition` 时间为 `2s`。
*   **`CardEditor.vue`**: 适配 `background` 和 `tags` 的列表编辑 UI。
